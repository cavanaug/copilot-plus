<task type="auto" tdd="true">
  <name>Task 2: GREEN — Implement the copilot wrapper script</name>
  <files>copilot</files>
  <action>
    Create `copilot` (no extension) in the repo root. Must be chmod +x.

    Script structure:
    ```bash
    #!/usr/bin/env bash
    set -euo pipefail
    ```

    **Step 1 — Locate real copilot binary:**
    Hard-code `REAL_COPILOT=/home/linuxbrew/.linuxbrew/bin/copilot`.
    (This is the known real binary path; exec at the end must call this absolute path, not rely on PATH lookup which would recurse back to this wrapper.)

    **Step 2 — Read and validate config:**
    ```bash
    CONFIG_FILE="${HOME}/.copilot/config.json"
    config_flags=()

    if [[ -f "$CONFIG_FILE" ]] && [[ -r "$CONFIG_FILE" ]]; then
      # Validate JSON
      if ! jq empty "$CONFIG_FILE" 2>/dev/null; then
        echo "copilot-wrapper: invalid JSON in $CONFIG_FILE" >&2
        exit 1
      fi
      # ... parse mappings (Step 3)
    fi
    # If file missing/unreadable: config_flags stays empty, proceed silently
    ```

    **Step 3 — Parse hardcoded field-to-flag mappings:**
    Define an associative array (bash 4+):
    ```bash
    declare -A FIELD_MAP=(
      [allowed_dirs]="--add-dir"
      [allowed_tools]="--allow-tool"
      [denied_tools]="--deny-tool"
      [allowed_urls]="--allow-url"
      [denied_urls]="--deny-url"
    )
    ```

    For each key in FIELD_MAP:
    - Use `jq -e 'has("KEY")' "$CONFIG_FILE"` to check if the field exists in the JSON
    - If it exists, check it is an array: `jq -e '.KEY | type == "array"' "$CONFIG_FILE"`
    - If NOT an array: `echo "copilot-wrapper: '$KEY' must be an array in $CONFIG_FILE" >&2; exit 1`
    - If an array: iterate with `jq -r '.KEY[]' "$CONFIG_FILE"` and append `"$FLAG" "$value"` pairs to `config_flags`
    - Keys not in FIELD_MAP are ignored (loop only iterates over FIELD_MAP keys)

    **Step 4 — Exec the real binary:**
    ```bash
    exec "$REAL_COPILOT" "${config_flags[@]}" "$@"
    ```
    `config_flags` is prepended; `"$@"` (user args) appended. `exec` replaces the wrapper process.

    After writing the script, run `chmod +x copilot`.

    Run `bats tests/copilot.bats` — ALL tests must PASS (GREEN phase).
